# 新規タブ用webページ

## 概要

ブラウザの「新規タブ」として使用する、自分専用のポータルサイトを作りたい。
ブックマーク機能やクッキー（Cookie）は使わず、ブラウザ内LocalStorageを利用してリンクを管理する。

## デザイン・レイアウト

- コンセプト: FireFoxの新規タブ（FireFox Home）風
- 配置: 
  - 中央上部: 大きめのデジタル時計（`16:00` 形式、リアルタイム更新）。
  - 時計の下: 検索バー（ページ読み込み時に自動フォーカス）。
  - 中央から下部にかけて: ショートカットのタイル（グリッドレイアウト）。
- タイルデザイン: 
  - 上部にFavicon（API利用）、下部にサイト名。
  - ホバー時に削除用の「×」ボタンを表示。
  - 最後に「＋（追加）」ボタンを配置。

## 機能要件

- 検索バー: Enterキーで、検索結果のページへ遷移
- ショートカット管理: 
  - 追加: 「+」ボタン押下で、URLとサイト名を入力するモーダルを表示
  - 保存: データをJSON形式でLocalStorageに格納
  - 削除: リストから対象を削除し、LocalStorageを更新

## 技術スタック

- ブラウザ: FireFoxを想定
- 言語: HTML5/CSS3/JavaScript(Vanilla JS)
- 外部ライブラリ: 不使用、または最小限
- ファビコン: API（`https://www.google.com/s2/favicons?domain=`）を利用

## 補足事項

- Github Pagesで公開できるような静的ページ。
- ファビコン取得に失敗した場合のデフォルト表示を考慮する。
- 新規タブとして開いたときに、検索バーに自動でフォーカスが当たっている状態にする。
